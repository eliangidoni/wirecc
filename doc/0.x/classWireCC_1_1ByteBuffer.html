Detailed Description

A byte buffer for reading and writing binary data.

Provides methods for serializing and deserializing various data types in big-endian format. Must call load() or clear() before use.

Warning
Mixing byte reads and writes is not supported.
Assumes two's complement representation for signed numbers.
